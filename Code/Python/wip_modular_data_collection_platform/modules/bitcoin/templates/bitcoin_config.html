Bitcoin Explorer — configuration

The explorer is stateless; these settings just point it at an API and tune display. The default targets Blockstream's public mainnet API. Swap in a testnet URL or your own self-hosted Esplora node to keep lookups private.

Common endpoints — Blockstream mainnet https://blockstream.info/api, testnet https://blockstream.info/testnet/api, or any Esplora instance http://your-node:3002.
