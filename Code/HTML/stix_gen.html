Malware objects represent malicious code and software used in cyber attacks.

Attack Pattern

Attack Patterns describe ways that adversaries attempt to compromise targets.

Campaign

Campaigns group adversarial behaviors that occur over time against specific targets.

Threat Actor

Threat Actors represent individuals, groups, or organizations believed to be operating with malicious intent.

Course of Action

Courses of Action are actions taken to prevent or respond to attacks.

Report

Reports are collections of threat intelligence focused on one or more topics.

Relationship

Relationships link STIX objects to describe how they are related to each other.

Sighting

Sightings denote that something in CTI was seen.

Observed Data

Observed Data conveys information about cyber security related entities such as files, systems, and networks.
